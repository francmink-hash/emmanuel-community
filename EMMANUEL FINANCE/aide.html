Bienvenue dans l'aide

🔐 Créer un compte utilisateur

Sur la page de connexion, cliquez sur Inscription, saisissez votre nom, un mot de passe et choisissez une question de sécurité. Vos données sont entièrement privées et cloisonnées.

🏦 Créer un compte financier

Rendez-vous sur la page Comptes, remplissez le nom, l'icône et la couleur, puis créez votre nouvelle caisse.

🏷️ Créer une catégorie

Depuis la page Comptes, ajoutez une catégorie et choisissez un groupe pour mieux classer vos dépenses.

➕ Enregistrer une transaction

Allez dans Transactions, remplissez le libellé, le montant, la date, le compte et la catégorie puis appuyez sur Entrée ou Sortie selon le type de flux.

🚀 Créer un projet d'épargne

Sur la page Projets, définissez un objectif, une échéance et un compte lié. L'application calcule l'effort quotidien pour atteindre l'objectif.

🤝 Prêts et emprunts

Sur la page Dettes, enregistrez un prêt ou un emprunt. Ajoutez des remboursements pour suivre le solde restant.

🔒 Confidentialité des données

Chaque utilisateur possède un espace totalement cloisonné. Vos comptes, transactions, projets et dettes ne sont jamais visibles par d'autres utilisateurs.

💾 Sauvegarde

Les données sont stockées localement dans votre navigateur. Faites une sauvegarde manuelle en exportant le fichier JSON si vous changez d'appareil.

FAQ

Mes données sont-elles partagées avec d'autres utilisateurs ?

Non. Chaque utilisateur a son propre espace étanche. Personne d'autre ne peut voir vos enregistrements.

Puis-je utiliser l'application sur un autre PC ?

Oui. Importez votre fichier JSON sur le nouveau PC pour restaurer vos informations.

Comment sauvegarder mes données ?

Utilisez le bouton d'export JSON disponible depuis la page principale. Le fichier contient votre profil et toutes vos données financières.

Que se passe-t-il si je vide mon navigateur ?

Les données seront perdues si vous n'avez pas sauvegardé. Pensez à exporter régulièrement votre caisse.

J'ai oublié mon mot de passe, que faire ?

Sur la page de connexion, cliquez sur Mot de passe oublié ? et répondez à votre question de sécurité pour réinitialiser votre accès.
